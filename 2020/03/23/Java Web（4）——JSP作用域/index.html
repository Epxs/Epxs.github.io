Java Web(4)——JSP作用域

发表于 2020-03-23 | 分类于 Java Web

JSP 域对象分别：page，request，session，application。每个域对象分别对应各自的作用域。

page 作用域

page 作用域只在当前的 jsp 页面有效，也就是用户请求的页面。其对应的内置对象是 pageContext（不是page）。

session 作用域

session 作用域表示在一次会话中有效。其对应的内置对象为 session。通过 session 为属性设定值，在一次会话期间，发起多次请求，都可以取出数据。

application 作用域

application 作用域在所有的 Web 应用程序中都有效，也就是当服务器开启到关闭这段时间，application 作用域中存储的数据都是有效的。同样可以通过 setAttribute 赋值和 getAttribute 取值。
