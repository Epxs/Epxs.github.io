Java Web(5)——EL表达式与JSTL

发表于 2020-03-25 | 分类于 Java Web

EL 表达式和 JSTL 标签库是提高 Web 开发效率的重要工具。

EL表达式

EL表达式的全称为 Expression Language，用于简化 JSP 页面中获取数据时的复杂代码。主要有以下作用：

  • 获取数据
  • 执行逻辑运算

使用 EL 表达式可以快速取出 JSP 域对象中的数据，通过setAtrribute 方法为域对象设置了属性之后，使用 EL 表达式可简化 getAtrribute 的步骤。

  • EL 表达式的基本语法：${变量名}，变量名为 setAtrribute(String key,Object o) 中的 key 值。

  • EL 表达式的查找数据的顺序：pageContext->request->session->application。若都未找到，则返回空。

JSTL

JSP 标准标签库（JSTL）是一个 JSP 标签集合，它封装了 JSP 应用的通用核心功能。JSTL 为开发者提供的⼀系列的标签，使⽤这些标签可以完成逻辑处理，如迭代，条件判断等操作。

优点：使用 JSTL 可以使代码更加简洁，能减少 JSP 脚本穿插的情况。实际开发中 EL 和 JSTL 结合起来使⽤，JSTL 侧重于逻辑处理，EL 表达式负责展示数据。

JSTL 的导入：

1 在 web/WEB-INF 下新建一个 lib 文件夹，将所需 jar 包（jstl.jar和standard.jar）导入到 lib 文件下。

2 引入 library。

3、在 JSP 页面头部引⼊ JSTL 命名空间，以核心标签库为例。
