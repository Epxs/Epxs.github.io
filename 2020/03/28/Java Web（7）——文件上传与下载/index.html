FileUpload 文件上传

FileUpload 是 apache 提供的一组用于文件上传下载的组件，使用前需引入两个 jar 包：

  • commons-fileUpload.jar
  • commons-io.jar

FileUpload 组件使用的过程大致分为四步：

  • 创建 DiskFileItemFactory 对象。
  • 通过 DiskFileItemFactory 获取 ServletFileUpload 对象。
  • 通过 ServletFileUpload 获取 FileItem 对象（集合）。
  • 解析 FileItem 对象。

FileItem 集合中包含了表单中的所有字段，可通过 isFormField 方法判断当前对象是否为文件类型。
