Spring MVC(4)——表单标签库

发表于 2020-04-17 | 分类于 Spring MVC

在渲染视图时， Spring 提供了一组全面的自动绑定数据的标签来处理表单元素。

Spring 标签库会渲染成 HTML 表单标签，且可以绑定 Model 中的属性。使用 Spring 表单表单标签库时，首先需在 JSP 页面声明如下指令。

Spring 表单标签库有以下常用标签：

form 标签

Spring form 标签会渲染成一个 HTML form，可以为其内部标签暴露绑定路径，用于数据绑定。可设置属性 modelAttribute 的值，为其绑定一个 Java Bean 对象。

form 标签一般会作为父节点，与其他表单标签（如 input 等）结合使用，将 model 中的数据展示在 JSP 页面。

input 标签

input 标签会渲染成一个 type = “text” 的 HTML input 标签。设定 path 属性的值，可以将绑定到 Java Bean 的某个属性。

checkbox 和 radiobutton 标签

form: checkbox 标签会渲染成一个复选框，checkbox 可以绑定三种类型，分别是：

  • 基本类型：如果属性的值等于 value 时，则选中。
  • boolean：当值为 true 时，表示选中，否则不选中。
  • 数组或集合：当绑定一个数组或集合时，如果数组或集合中的元素等于 value 的值，则选中。

form: radiobutton 标签会渲染成一个单选框，如果属性的值和 value 相等时，则选中。

checkbox 和 radiobutton 都可以通过设置 path 来绑定某个属性。
