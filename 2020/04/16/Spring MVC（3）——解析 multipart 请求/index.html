Spring MVC(3)——处理 multipart 请求

发表于 2020-04-16 | 分类于 Spring MVC

Web 应用中，处理用户上传的文件是很常用的需求，Spring MVC 框架提供了一套处理 multipart 请求的机制。

单文件上传

Spring MVC 提供了两个 multipart 解析器：

  • CommonsMultipartResolver：基于 FileUpload 组件，Spring MVC 对其进行了封装，使用之前，需要导入 jar 包。
  • StandardServletMultipartResolver：依赖于 Serlvet 3.0 对 multipart 请求的支持，始于 Spring 3.1。使用时需要将应用部署到支持 Serlvet 3.0 的容器中。

Spring MVC 中处理用户上传的文件时，只需在控制器方法中，接收一个 MultipartFile 对象，将这个对象进行解析即可。
