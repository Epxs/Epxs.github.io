Mybatis(4)——动态SQL

发表于 2020-04-27 | 分类于 Mybatis

支持动态 SQL 是 Mybatis 的强大特性，动态即可以 SQL 语句可以根据条件的不同而改变，大大提高了 SQL 语句的灵活性。

动态 SQL

动态 SQL 是 Mybatis 的强大特性之一。可以帮助我们根据不同条件拼接 SQL 语句，实现对 SQL 的灵活操作。Mybatis 通过表达式进行判断，对 SQL 进行灵活拼接、组装。

主要通过以下标签：

  • if
  • where
  • choose（when，otherwise）
  • set
  • foreach
